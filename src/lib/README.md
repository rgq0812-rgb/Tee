# 🏌️‍♂️ ONYX V2 - Tactical Golf

**L'élite de l'assistance tactique pour le golf.**  
ONYX V2 n'est pas une simple application de score, c'est un système de liaison tactique complet intégrant GPS live, intelligence artificielle, gestion d'arsenal et détection gestuelle.

---

## 🚀 Fonctionnalités Clés

### 🧠 Liaison Caddie IA (Système ADAM/ANTONI/ARNOLD)
Recevez des conseils tactiques personnalisés basés sur votre position réelle, le vent, et votre profil de jeu.
- **ADAM (Stratégie pure)** : Impérial et sobre, privilégie la sécurité.
- **ANTONI (Analyste poétique)** : Technique et nuancé.
- **ARNOLD (Agressif)** : Direct, cible les drapeaux sans vergogne.

### 🎒 Gestion de l'Arsenal
Configurez votre sac de golf complet.
- Enregistrez vos distances moyennes pour chaque club (Driver, Fers, Wedges, Putter).
- L'IA utilise ces données pour vous suggérer le club parfait lors de chaque approche.

### 💎 État du "Toucher" (Player Form)
Parce que le golf est une question de sensation, ONYX s'adapte à votre état :
- **❄️ FROID** : Stratégie prudente, clubs plus tolérants.
- **⚡ FORME** : Jeu standard équilibré.
- **💎 PURE** : Toucher exceptionnel, l'IA vous pousse à l'agressivité.

### 👂 Liaison Tactique "Ear" (Gestuelle)
Activez le mode **Ear Active** pour une liaison mains-libres. 
- Portez votre téléphone à votre oreille pour déclencher automatiquement le rapport du caddie IA.
- Une vibration confirme la liaison établie.

### 🛰️ GPS & Télémètre Tactique
- Calcul de distance en temps réel entre votre position et le centre du green (ou la cible).
- Vue satellite tactique de chaque trou avec zones d'atterrissage.
- Gestion du vent live (vitesse et direction).

### 📈 Progression Élite & Handicap
- **Index de jeu dynamique** : Réglez votre handicap (0 à 54) dans votre profil pour que l'IA adapte ses conseils à votre niveau réel.
- **XP Élite** : Gagnez des points d'expérience lors de vos parties pour monter en grade.

---

## 🛠️ Configuration

1. **Localisation** : Autorisez l'accès au GPS pour le calcul des distances.
2. **Capteurs** : Dans le menu Sync, activez la "Liaison Tactique Ear" pour utiliser les capteurs de mouvement.
3. **Arsenal** : Allez dans votre **Profil** pour configurer les distances de vos clubs.

---

## 💻 Tech Stack

- **Framework** : React 18 + Vite
- **Animations** : Motion (framer-motion)
- **Intelligence Artificielle** : Google Gemini API (Modèle Flash 2.0 pour des réponses instantanées)
- **Database/Auth** : Firebase (Firestore & Auth)
- **Styles** : Tailwind CSS
- **Icons** : Lucide React
